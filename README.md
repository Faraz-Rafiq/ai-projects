# AI Project —

A 4-module Artificial Intelligence project implementing core AI concepts
from search algorithms to machine learning.

## Module 1 — Intelligent Urban Delivery Robot
Path planning for a delivery robot on a 15x15 grid using:
BFS, DFS, UCS, Greedy Best First Search, and A*
Compares algorithms on path cost, execution time, and nodes explored.
Tech: Python

## Module 2 — Sudoku CSP Solver
Sudoku puzzle solver formulated as a Constraint Satisfaction Problem.
Implements AC-3 (Arc Consistency) and Backtracking algorithms with a
Tkinter GUI. Supports Easy, Medium, and Hard puzzles with hint system.
Tech: Python, Tkinter

## Module 3 — Automated Exam Management System
K-Means clustering to automate seating plans and faculty allocation
for 2400+ students across 5 departments and 4 batches in 30 exam rooms.
Tech: Python, NumPy, Pandas, Scikit-learn

## Module 4 — Iris Flower Classifier
Implements Perceptron Learning Rule and Gradient Descent Delta Rule
from scratch to classify iris species. Compares activation functions
and learning rates using 80/20 train-test split.
Tech: Python, UCI Iris Dataset

## Tech Stack
Python | Tkinter | NumPy | Pandas | Scikit-learn
